For this project I had to create my own photoshop tool using the p5.js library. Here are the controls for it.

1 - Thin black line
2 - Constant black lines
3 - Yellow spray line
4 - Red thick line
5 - Constant red squares
6 - Constant rectangle outline
7 - Constant small green squares
8 - Constant small pink squares
9 - Constant medium orange squares
0 - Constant big random color rectangles
g - Constant HELP text
c - Constant random color circles
s - Tan line
b - Light blue line
x - Clear canvas
p - Take screenshot
